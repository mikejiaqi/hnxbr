使用说明
1. 数据库概述

本应用是一个工业油漆数据库，用于存储和查询各种油漆产品的技术参数。所有数据存储在浏览器本地，无需服务器支持。

2. 基本功能
  • 浏览数据：在"数据库"页面查看所有油漆产品
  • 搜索产品：使用搜索框输入关键词查找产品
  • 添加产品：点击"添加产品"导航项，填写表单添加新产品
  • 编辑/删除：在产品列表中使用操作按钮编辑或删除产品
  • 导入/导出：在"导入/导出"页面备份或恢复数据
3. 自动计算功能

本应用提供强大的自动计算功能，您只需输入以下核心参数：

  • 干膜厚度(μm)
  • 固体含量(%)
  • 比重
  • 损耗系数
  • 涂装总面积(㎡)（可选，用于计算涂装用量）
  • 单位价格(元/L)（可选，用于计算成本）

系统将自动计算以下结果：

  • 理论材料耗量(㎡/L) = 固体含量 × 10 / 干膜厚度
  • 理论材料耗量(㎡/kg) = 理论材料耗量(㎡/L) / 比重
  • 实际材料耗量(㎡/L) = 理论材料耗量(㎡/L) / 损耗系数
  • 实际材料耗量(㎡/kg) = 理论材料耗量(㎡/kg) / 损耗系数
  • 涂装用量(L) = 涂装总面积 / 实际材料耗量(㎡/L)
  • 涂装用量(kg) = 涂装总面积 / 实际材料耗量(㎡/kg)
  • 总成本(元) = 涂装用量(L) × 单位价格(元/L)
  • 单位价格(元/㎡) = 单位价格(元/L) / 实际材料耗量(㎡/L)
4. 数据字段说明
字段名称 说明
产品名称 油漆的名称，必填项
品牌 油漆的品牌或制造商
类型 油漆的类型，如底漆、中间漆、面漆等
颜色 油漆的颜色
干膜厚度(μm) 涂层干燥后的厚度，核心参数，必填项
固体含量(%) 油漆中固体成分的百分比，核心参数，必填项，0-100之间
比重 油漆的密度比值，核心参数，必填项，大于0
涂刷道数 需要涂刷的次数
施工方式 油漆的应用方法
损耗系数 实际施工中的材料损耗系数，核心参数，必填项，大于等于1
理论材料耗量(㎡/L) 理论上每升油漆可涂刷的面积，自动计算
理论材料耗量(㎡/kg) 理论上每千克油漆可涂刷的面积，自动计算
实际材料耗量(㎡/L) 考虑损耗后每升油漆可涂刷的面积，自动计算
实际材料耗量(㎡/kg) 考虑损耗后每千克油漆可涂刷的面积，自动计算
涂装总面积(㎡) 需要涂装的总面积，用于计算涂装用量
涂装用量(L) 完成涂装所需的油漆体积，自动计算
涂装用量(kg) 完成涂装所需的油漆重量，自动计算
单位价格(元/L) 每升油漆的价格
单位价格(元/kg) 每千克油漆的价格
单位价格(元/㎡) 每平方米涂装的价格，自动计算
总成本(元) 完成涂装的总成本，自动计算
备注 其他相关信息
5. 数据存储

所有数据存储在浏览器的本地存储中，不会上传到任何服务器。请定期导出备份您的数据，以防浏览器缓存被清除导致数据丢失。

6. 常见问题

只需输入核心参数（干膜厚度、固体含量、比重、损耗系数），系统会自动计算理论和实际材料耗量。如果再输入涂装总面积和单位价格，系统还会自动计算涂装用量和总成本。

是的，所有添加、编辑和删除操作都会自动保存到浏览器的本地存储中。

您可以在一台设备上导出数据为JSON或CSV文件，然后在另一台设备上导入该文件。
